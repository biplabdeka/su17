Overview

This programming assignment has two parts:

  1. Part A: You will design and implement a simple, interactive card using only HTML, CSS, and Javascript. The webpage should mirror the layout and funcationaly shown in the examples. This assignment is meant to be a very humble introduction to HTML5, CSS, and JavaScript.
  2. Part B: You will be setting up your development environment as detailed in Dev Lab 1. Getting this right is important and would save you from a ton of hassle in all the future MPs.

Required Features

Your webpage will have to implement the features listed below. Note that the examples are only meant to illustrate the features. You do not have to make you webpage look similar to the examples. You just have to incorporate that feature in some form.

  • Part I: You will be using Codepen for this part. You should create a new pen on codepen.
    1. Elements: You should use at least one of all the following: image, link, paragraph, heading.
    2. Layout: Should closely mirror the example. Everything should be centered horizontally(card, image, text, heading)
    3. Styling: All styling must lie within CSS section
    4. JavaScript: All your code should lie in the JS section
    5. Interactions: Two interactions namely: Clicking the title/image to show/hide details, and changing color on button press need to be present. You should have at least 3 buttons to switch colors.
    6. The card must occupy 80% of the page height when details are shown, and 30% when details are hidden. The image should have a height-width of 150px

  • Part II: You should follow instructions for Lab1 and MP1.
    1. Get MP1 starter to run
    2. Change SASS inside the source_sass folder so that Hello World is centered
    3. Add SASS to change the background color
  • Attention: Any changes in in the public folder will not stick.
